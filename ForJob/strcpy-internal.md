#### 看strcpy源码！

看了strcpy的源码实现，实现思路是差不多的。
```C
/* Copy SRC to DEST.  */
char *
strcpy (char *dest, const char *src)
{
  char c;
  char *s = (char *) src;
  const ptrdiff_t off = dest - s - 1;

  do
    {
      c = *s++;
      s[off] = c;
    }
  while (c != '\0');

  return dest;
}
```

#### 为何glibc中采用s[off]的方式来复制字符？

采用 s[off] = *s++ 而不是 *d++ = *ss 可以性能更优，因为只需要移动一个指针。
